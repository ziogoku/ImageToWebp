# ImageToWebP 🦀

A high-performance command-line tool written in Rust designed to convert images (PNG, JPG, BMP) to WebP format efficiently. 

## 🚀 Why this project?
WebP is the gold standard for web performance. I built this tool to simplify the workflow of developers and designers by enabling batch conversion of entire directories, ensuring smaller file sizes without compromising visual quality.

## ⚙️ Features
- **Multi-format Support:** Seamlessly converts PNG, JPG, and BMP images.
- **Fast:** Powered by Rust for blazing-fast execution speeds.
- **Batch Processing:** Handle large directories with just one command.
- **Web Optimized:** Generates ready-to-use assets for modern websites.

## 📦 Installation
Make sure you have [Rust/Cargo](https://rustup.rs/) installed.

```bash
git clone [https://github.com/your-username/ImageToWebP.git](https://github.com/your-username/ImageToWebP.git)
cd ImageToWebP
cargo build --release
